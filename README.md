# Super Metroid Randomap Web Generator

An automated Python script to generate randomized Super Metroid ROMs using the [maprando.com](https://maprando.com) service.

## Description

This script automates the process of generating randomized Super Metroid ROMs through the maprando.com web interface using Selenium WebDriver. It allows you to:

- Generate one or more randomized seeds in batch mode
- Customize Samus sprites and E-Tank colors
- Use custom or predefined configuration presets
- Automatically download the generated ROMs

## Requirements

- Python 3.x
- Selenium (`pip install selenium`)
- [ChromeDriver](https://googlechromelabs.github.io/chrome-for-testing/) or [GeckoDriver](https://github.com/mozilla/geckodriver/releases) (installed and accessible in your system's PATH)
- An original Super Metroid ROM file

## Usage

### Basic Syntax

```bash
python maprandogen.py -i <path_to_original_rom> [options]
```

### Parameters

#### Required

- `-i, --input`: Path to the original Super Metroid ROM (required)

#### Optional

- `-o, --output`: Output directory for the generated ROMs (default: current directory)
- `-g, --generate`: Number of seeds to generate (default: 1)
- `-p, --preset`: A preset JSON file or a preset name
- `-r, --random`: Randomize the Samus sprite and E-Tank color
- `-s, --sprite`: Specify a specific sprite for Samus
- `-c, --color`: Specify a specific color for the E-Tanks (hex code)
- `-b, --browser`: Browser to use (default: chrome)
- `-t, --token`: Seed ID or URL to unlock spoiler map
- `-l, --list`: Show list of all available sprites and colors

### Usage Examples

#### Generate a single seed with default settings

```bash
python maprandogen.py -i "Super Metroid.sfc"
```

#### Generate 5 seeds with randomized sprites and colors

```bash
python maprandogen.py -i "Super Metroid.sfc" -g 5 -r
```

#### Generate with a custom preset and output to a specific folder

```bash
python maprandogen.py -i "Super Metroid.sfc" -p "Tournament" -o generated_roms
```
#### Generate with a custom sprite and color

```bash
python maprandogen.py -i "Super Metroid.sfc" -s dread_samus -c 38ded3
```
#### Unlock Spoiler Map

```bash
python maprandogen.py -t MBLmpWfwg
```

## Customization

### Sprites and E-Tank Colors

The script supports numerous different sprites and colors for E-Tanks. To see all available sprites, visit the [sprite gallery on GitHub](https://github.com/blkerby/MapRandoSprites/tree/7efba2d509e27c36666f72e949422418b4cc5dd3/samus_sprites).

### Other Customizations

You can modify additional game values by editing the `other_values` dictionary in the `maprandogen.py` file, which includes options such as:

- Custom controls (button mapping)
- Visual themes (doors, rooms, palettes)
- Gameplay options (moonwalk, quick reload, spin lock)
- Visual effects (flashing, shaking)
- HUD settings (reserve tanks, room names)

## Presets

Presets allow you to configure various game settings, such as:

- Randomizer difficulty and logic
- Quality-of-life options
- Map layout

You can use:
1.  **Predefined presets** from the maprando.com website by specifying the name.
2.  **Custom JSON files** by creating your own preset on the site and exporting it.

## Output

The generated ROMs are saved with the following format:

```
Super Metroid Randomap <seed_id>.sfc
```

Where `<seed_id>` is the unique seed identifier generated by maprando.com.
